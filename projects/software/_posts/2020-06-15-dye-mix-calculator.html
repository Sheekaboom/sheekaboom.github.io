Color Mixer UNDER CONSTRUCTION

This is a color mixing tool. Like many others, this tool allows you to to add a set of input colors and add/remove amounts of the color to get a desired mixture color. Unlike other tools, this tool also allows you to specify a desired output color and will calculate the required mixture of the colors you have to get a specific output.

Instructions

Adding Colors:

  1. Click the 'Add' Button
  2. Set the name of the Color
  3. Click on the color in the table to open the color selector
  4. Select the color from the wheel or by value
  5. Click 'Save' after all colors have been added to store for next time

Mixing Colors:

  1. In the table of available colors, change the value of 'Count'
  2. Add or remove amounts of each color until the final color is seen in the 'Mixed Color' box

Calculating colors

  1. Click on the 'Desired Color' color box
  2. Select the color by wheel or value
  3. Click 'Calculate' to automatically populate the 'Count' in the table
